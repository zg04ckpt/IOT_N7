# BTL - IOT - Nhóm 7

## Cấu trúc
- `backend`: API server (Node.js)
- `front-end-web`: Web client (React + Vite)
- `local/desktop-app`: Desktop app (Python)

- `local/firmwares`: Firmware chính cho ESP32C3 và ESP32CAM
- `mobile-app`: Mobile app (React Native)
- `ai-train-result`: Kết quả quá trình train tăng cường AI model
